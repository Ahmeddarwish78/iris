🌸 Iris Flower Classification

This project analyzes the Iris dataset using Python to explore and classify different flower species based on their features such as sepal length, sepal width, petal length, and petal width.

🛠 Tools Used

Python, Pandas, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook

📌 Project Features
Data exploration (EDA)
Data visualization
Correlation analysis
Machine learning classification
Model evaluation and prediction


🎯 Goal
Build a classification model to accurately predict Iris flower species from their measurements.
